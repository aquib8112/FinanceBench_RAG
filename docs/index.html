FinanceBench RAG

⚠️ Please read this before using ⚠️

This AI system answers questions using a fixed set of 84 FinanceBench source documents, covering 10-K filings, 10-Q filings, 8-K reports, and earnings-call transcripts. In internal evaluations, it correctly identified the exact supporting document in around 70% of test cases. You may review the full document list before asking questions.

This project is designed as a scaled-down prototype of a retrieval-and-reasoning pipeline that can be expanded to handle substantially larger collections of financial documents with minimal architectural changes.

This is an active work in progress, and the system’s accuracy and reliability are being steadily improved.

Thank you for trying it out.
